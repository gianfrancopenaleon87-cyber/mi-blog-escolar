Hola Mundo y Variables

Python es famoso por ser legible. Mira lo fácil que es crear un mensaje:

# Esto es una variable de texto
mensaje = "¡Bienvenido a mi blog!"
print(mensaje)
> ¡Bienvenido a mi blog!

Matemáticas Rápidas

Puedes usar Python como una calculadora súper potente.

numero1 = 10
numero2 = 5
resultado = numero1 + numero2
print(resultado)
> 15
